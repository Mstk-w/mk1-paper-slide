# 言語ルール (Language Rules)

**最重要**: 本プロジェクトにおけるコミュニケーションおよび成果物の第一言語は「日本語」です。

## 1. 基本方針
- **ユーザー対話**: 全て日本語で行います。
- **生成コンテンツ**: スライド内のテキスト、検証レポート、計画書など、全てのユーザー向け成果物は日本語で記述します。
- **思考プロセス**: AIの思考プロセス（Chain of Thought）自体は英語でも構いませんが、最終的な出力は日本語でなければなりません。

## 2. コード・開発関連
- **コメント**: コード内の docstring および行内コメントは、ユーザーが保守しやすいように**日本語**で記述してください。
- **コミットメッセージ**: Gitのコミットメッセージや変更内容の説明も日本語で行います。
- **例外**:
    - プログラミング言語の予約語（def, class, import 等）
    - 一般的なライブラリ名（python-pptx, pandas 等）
    - 変数名・関数名は、Pythonの慣習に従い英語（スネークケース）とします（例: create_slide, slide_width）。ただし、その意味が曖昧になる場合はコメントで補足してください。
